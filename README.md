# FlexUI

O FlexUI é um framework front-end de JavaScript puro (Vanilla JS), desenvolvido sem dependências externas de runtime, com o propósito de criar interfaces de usuário (UI) complexas, modulares e reconfiguráveis. O seu objetivo principal é permitir a construção de layouts dinâmicos baseados em painéis (widgets), linhas e colunas, permitindo que o usuário final reorganize, redimensione e gerencie esses painéis de forma fluida, similar ao que é encontrado em IDEs (Ambientes de Desenvolvimento Integrado) ou ferramentas de BI (Business Intelligence).
